# LeilãoVision

App web para análise e acompanhamento de leilões de imóveis.

## Visit Website

https://leilao-vision.vercel.app/login

## Stack

- Next.js 14 (App Router)
- React 18 + TypeScript strict
- Tailwind CSS
- Supabase (Auth, PostgreSQL, Storage, RLS)
- Recharts

## Módulos

- Dashboard
- Imóveis
- Leilões
- Análise
- Documentos
- Tarefas e Prazos
- Lances
- Pós-Leilão
- Contatos
- Relatórios
